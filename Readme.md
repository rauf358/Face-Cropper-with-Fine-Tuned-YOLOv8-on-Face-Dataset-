💥 Face Cropper with Fine-Tuned YOLOv8 on Face Dataset 

This project implements a Face Cropper using a fine-tuned YOLOv8 model trained on a curated face detection dataset. 
The model accurately detects and localizes faces in images or video streams, enabling precise cropping of facial regions.
By leveraging the lightweight yet powerful YOLOv8 architecture, the system ensures real-time performance and high accuracy, making it suitable for applications like
facial recognition preprocessing, dataset generation, and surveillance systems. This Project is capable of handling live streams.
